Generated protobuf C++ bindings. Do not edit by hand.

consumers of session_cpp do not need protoc at build time.

Regenerate after editing any .proto with one of:
    cmake -B build -DSESSION_REGEN_PROTO=ON
    cmake --build build --target regen_proto
or
    ../bash/gen_proto.sh
